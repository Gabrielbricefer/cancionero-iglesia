# 🎵 Cancionero para Iglesia

Sistema web para gestionar canciones con acordes para guitarra, diseñado para grupos musicales de iglesia.

## ✨ Características

- 📝 **Gestión de canciones**: Crear, editar y eliminar canciones con acordes
- 🎸 **Acordes inteligentes**: Soporte para todos los tipos de acordes (m, 7, maj7, aug, dim, sus, etc.)
- 🔄 **Transposición**: Subir/bajar tonos (1 tono o 1/2 tono)
- 🔀 **Conversión de notación**: Cambiar entre cifrado latino (Do, Re, Mi) y americano (C, D, E)
- 📂 **Categorización**: Orden de misa y tiempo litúrgico
- 📋 **Planificación de misa**: Crear listas de canciones y compartir mediante enlace
- 🔐 **Autenticación**: Registro e inicio de sesión de usuarios
- 🎯 **Modo "Orig"**: Restaurar canción a su estado original

## 🛠️ Tecnologías

- **Backend**: Python + Flask
- **Frontend**: HTML, CSS, Bootstrap
- **Base de datos**: SQLite
- **JavaScript**: Lógica de transposición y acordes
